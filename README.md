ESG Performance & Predictive Analytics — S&P 500 (2014–2022)

Project Overview

This project analyzes ESG performance trends of S&P 500 firms between 2014 and 2022 using Python. The analysis explores ESG pillar dynamics, firm segmentation, and predictive modeling of ESG improvement.

Objectives :
Examine multi-year ESG trends
Compare Environmental, Social, and Governance pillars
Segment firms using clustering
Develop a predictive model for ESG score improvement

Dataset:
S&P 500 Firms ESG Sustainability Reports Dataset

Variables used:
e_score
s_score
g_score
total_score
ticker
year

Methods Used:
Exploratory Data Analysis (EDA)
Time Series Trend Analysis
K-Means Clustering
Custom Weighted ESG Ranking Model
Correlation Analysis
Random Forest Classification Model

Key Findings:
ESG performance peaked in 2018 and declined slightly afterward.
Social pillar scored highest on average; Environmental lowest.
Environmental performance was the strongest differentiator of ESG leaders.
Predictive model achieved 79% accuracy in forecasting ESG improvement.
